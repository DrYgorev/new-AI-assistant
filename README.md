# Описание проекта: ИИ-ассистент для компании по продаже БАДов

## Общая информация

В рамках данного проекта была разработана система на основе искусственного интеллекта, предназначенная для автоматизации взаимодействия с клиентами в сфере продаж биологически активных добавок (БАДов). Основной задачей ИИ-ассистента является осмысленное ведение диалога с клиентом, предоставление информации о продуктах, а также организация встреч и их планирование через Google Календарь.

## Основные функции ИИ-ассистента

### 1. **Осмысленное ведение диалога с помощью нейросети**
   - Ассистент способен понимать контекст запросов пользователей.
   - Обеспечивает естественный и плавный диалог, адаптируясь к стилю общения клиента.
   - Поддерживает многоуровневые вопросы и ответы, позволяя пользователям уточнять детали.

### 2. **Ответы на вопросы по базе знаний**
   - Использовано внутреннее хранилище данных о продуктах: состав, польза, рекомендации, противопоказания и т.д.
   - ИИ-ассистент извлекает точную информацию из заранее подготовленной базы знаний.
   - Все ответы соответствуют научно обоснованным данным и маркетинговым материалам компании.

### 3. **Запись клиента на встречу и добавление встречи в Google Календарь**
   - В случае необходимости переговоров с менеджером или консультантом, ИИ предлагает клиенту выбрать удобное время.
   - После подтверждения времени встречи, запись автоматически добавляется в Google Календарь как клиенту, так и представителю компании.
   - Система отправляет напоминания и ссылку для онлайн-встречи (при использовании Google Meet).

## Используемые технологии и сервисы

### 1. **Qwen**
   - Qwen был использован для создания системного промпта, который задаёт стиль, правила и цели поведения ИИ-ассистента.
   - Также Qwen применялся для генерации и пополнения базы знаний, что позволило значительно повысить точность и информативность ответов.

### 2. **Савви (Suvvy AI) — https://suvvy.ai/**
   - Для реализации ИИ-ассистента была выбрана платформа Suvvy AI, которая позволяет создавать мощных чат-ботов с возможностью работы по сложным сценариям.
   - Преимущества использования Савви:
     - Работа на базе надёжных LLM (Language Model Models).
     - 100% обработка обращений от пользователей.
     - Высокая точность ответов — свыше 95%.
     - Гибкая интеграция с CRM, мессенджерами и маркетплейсами.
     - Возможность расширения функционала под специфические задачи бизнеса.

## Интеграции

- **CRM-система**: все данные о клиентах, истории диалогов и записи встреч синхронизируются с CRM, обеспечивая комплексное управление клиентскими отношениями.
- **Мессенджеры**: ассистент доступен в популярных каналах коммуникации (Telegram, WhatsApp, Viber), что делает его легко доступным для целевой аудитории.
- **Google Календарь**: автоматизация записи на консультации и планирования встреч без участия человека.
